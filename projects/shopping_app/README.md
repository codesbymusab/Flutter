### Shopping App

A clean, multipage Flutter shopping app built as an introductory project to demonstrate common e‑commerce UI patterns and state management. It includes product browsing, filtering, a cart workflow, and a simple inventory model — ideal for learning Flutter fundamentals and prototyping mobile shopping experiences.

---

### Features
- **Filters** — Narrow product lists by category and attributes.  
- **Search** — Instant product search with live results.  
- **Cart** — Add, update, and remove items; view totals.  
- **Product Variants** — Support for size, color.

---

### Key Learnings
- **ListView.builder** for efficient, scrollable lists.  
- **SnackBar** for lightweight user feedback.  
- **AlertDialog** for confirmations and prompts.  
- **Chip** for compact filter UI and tags.  
- **State Management with Provider** for app-wide state and reactivity.  
- **JSON Parsing** for loading product data and variants.
